<h1>초기 세팅</h1>
<h2>Initialization</h2>   

```
git init
```
git start to view this directory
<br><br>

<h2>Add File</h2>

```
git add 파일명
```
save the file's current state<br>
(e.g. `git add './how to use Git.md'`)
<br><br>

<h2>File Memo</h2>

```
git commit -m '메모 내용'
```
memo about file<br>
(e.g. `git commit -m 'this is git manual`)<br>
if inquiry the history, this memo will help
